# Bochs命令

- s：单步调试

- s xxx：进行xxx次单步调试

  e.g. s 200：进行200次单步调试

- b：打断点，后面跟着内存地址

  e.g. b 0x7c00：在Boot Loader开始运行处打断点

- c：继续执行，直到出发断点

- q：退出Bochs

- r：查看通用寄存器

- sreg：查看段寄存器

- xp /nuf addr：查看内存

  n：查看多少个单位

  u：单位，u可以是b（字节）、h（2字节）、w（4字节）、g（8字节）

  f：输出格式，f可以是x（十六进制）、d（十进制）、u（无符号十进制）、o（八进制）、t（二进制）

  e.g. xp /1wx 0x7c0f1：查看1个word，以十六进制输出

  e.g. xp /1bx 0x7c0f1：查看1个byte，以十六进制输出
